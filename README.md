Here's an interactive website written entirely in HTML and CSS, and it's also my first project in this field.

There are four pages, the first one to open is the Index, and you can navigate to the others from there.

The site also works on mobile devices thanks to media queries.

This project is the result of a group effort, but one of our members had a mishap :( so it's not entirely finished.

This site is about Brazil in general, but that's not the main focus; it's the technical aspects that are interesting.

All images and fonts are available, do not change the file names.

Everything is freely accessible, and I will continue to update it ;D
